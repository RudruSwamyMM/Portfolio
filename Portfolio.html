Projects

  • WhatIfWorld : Powered by AI

    It is a web application providing AI driven answers for hypothetical questions on domains like "Education, Technology, Society, Environment, etc". An inbuilt AI model "Cerabras-Llama-3.3-70B" optimizes answers and validates in pre-defined format displaying "Domain, Probability, Key implications, Potential challenges, Statistical impact overview with bar and donut chart".

  • Smart Medicine Reminder

    It is an application focusing on alerting patients about medicines prescribed for them at regular intervals with details including "Time, dosage, etc" updated by the caretaker. The application comes with reminders as alarms, WhatsApp notifications with links. If the patient finishes medicines, they can update it in the application to notify the caretaker, otherwise the caretaker gets messages about missed doses.

  • BloomWatch

    It is a project offering flower bloom status of a particular region using NASA's NDVI dataset by generating a token. If a user selects a particular location on a map, it identifies the location by altitudes according to NASA satellite coordinates. If users want to preserve data for further use, they can store it by uploading local pictures and adding notes.
